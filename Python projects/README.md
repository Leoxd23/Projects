# Proyecto de Análisis Numérico y Algoritmos en Python

Esta carpeta reúne varios scripts de Python relacionados con análisis numérico, resolución de ecuaciones, comparación de algoritmos y simulación computacional. Cada archivo representa una práctica o ejercicio orientado a fortalecer la lógica matemática y la programación.

## Archivos incluidos

### 1. `Analisis Numérico.py`
- Descripción: script para resolver sistemas de ecuaciones lineales mediante eliminación gaussiana y ecuaciones no lineales mediante Newton-Raphson.
- Objetivo: mostrar métodos básicos de solución numérica para problemas algebraicos.

### 2. `Analisis Numérico 2.py`
- Descripción: implementación de métodos para encontrar raíces de ecuaciones, incluyendo Newton-Raphson, bisección y resolución simbólica con SymPy.
- Objetivo: comparar estrategias para localizar soluciones de funciones matemáticas.

### 3. `Analisis Numérico 3.py`
- Descripción: script que resuelve sistemas lineales con Gauss-Jordan y ecuaciones no lineales usando Newton-Raphson, bisección, regla falsa y punto fijo.
- Objetivo: ampliar el repertorio de métodos numéricos y mostrar distintos enfoques para un mismo problema.

### 4. `Analisis Numérico 4.py`
- Descripción: versión más robusta de un buscador de raíces. Detecta si la expresión es un polinomio y, si no lo es, intenta encontrar raíces numéricamente en un intervalo.
- Objetivo: demostrar una solución más avanzada y general para análisis numérico.

### 5. `EcSolver.py`
- Descripción: programa interactivo para resolver sistemas de ecuaciones lineales y ecuaciones no lineales mediante métodos numéricos.
- Objetivo: proporcionar una herramienta de práctica para cálculos matemáticos aplicados con Python.

### 6. `LRU.py`
- Descripción: simulación de caché con políticas FIFO y LRU, junto con comparación de fallos y tiempos de ejecución.
- Objetivo: mostrar conceptos de estructuras de datos y análisis de rendimiento en sistemas computacionales.

### 7. `montecarlo_pi.py`
- Descripción: implementación del método de Monte Carlo para aproximar el valor de $\pi$.
- Objetivo: demostrar una técnica probabilística y visualizar la relación entre número de iteraciones, error y tiempo de ejecución.

### 8. `CorrerEnCMD.py`
- Descripción: ejemplo de suma paralela usando `threading` para comparar el rendimiento con diferentes cantidades de hilos.
- Objetivo: ilustrar programación concurrente y análisis de tiempo de ejecución en paralelo.

## Tecnologías utilizadas

- Python
- NumPy
- SymPy
- Matplotlib
- Threading
- Programación matemática y algorítmica

## Objetivo general

Esta carpeta refleja trabajo práctico en cálculo numérico, desarrollo de algoritmos, análisis de rendimiento y programación aplicada con Python. Es una buena base para mostrar habilidades técnicas en un portafolio personal o en procesos de selección.

## Cómo ejecutar

Puedes ejecutar cualquiera de los scripts con Python desde la terminal, por ejemplo:

```bash
python "Analisis Numérico.py"
```

O con el archivo que desees revisar.
